## Run Locally

Clone the project

```bash
  git clone https://github.com/Avitohol1/weather_app_ts
```

Go to the project directory

```bash
  cd weather_app_ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
